**GreenCity QA Test Cases**
**Project Description**

This repository contains test cases and basic UI automation for the GreenCity web application.

The goal of this project is to demonstrate:

Manual testing skills
Test case design (ISTQB approach)
Basic test automation using C# and Selenium
**Test Coverage
Create News Feature**

Tested functionality includes:

Authorization (authorized / unauthorized user)
Title validation (required, max length)
Content validation
Image upload:
File format (PNG/JPG)
File size (<10MB)
Source URL validation:
Valid (http/https)
Invalid formats
Tags limitation (max 3)
